Webkriti is a theme based UI design contest for front-end developers.

Register

Pick up a team or participate individually by registering at :
http://goo.gl/forms/PKIObh9uqP

Theme

You have to create a front-end for an ecommerce website.
More precisely, you are free to choose some product or brand and create a front-end for their website.
For eg. visit http://www.nike.com/in/en_gb/c/cricket

Specifically you're required to create the following pages:-
  • A landing page(home page) for the website.
  • An info page (Similar to about us section).
  • A contact section.
  • An items showcase.(similar to www.adidas.co.uk/men-football-shoes).
PS: Feel free to add in more pages. The items showcase will display the products. You're NOT required to create a complete cart system/interface.

Rules and Instructions

  • There are no preliminary rounds.
  • Contestants can participate individually or in a team of two.
  • The theme for the event will be disclosed ,few hours prior to the start of the event.
  • The event judging will be done online.
  • Start with empty repository on github and commit the changes to the project.
  • Contestants have to host their work online (Eg.using gh-pages (github), codepen, jsfiddle ) and mail the links to iit2014156@iiita.ac.in and cc to irm2014004@iiita.ac.in. For more information on using gh-pages , follow the link - https://pages.github.com/.
  • Use of frameworks such as bootstrap, jquery,materialize CSS is allowed. You may refer to other templates,websites for reference. However, code duplication is prohibited and will lead to disqualification.
  • Contestants are required to make separate files for CSS, JS, jquery etc. for structured code.
  • Decision of the organisers is final and no changes will be entertained.

Judging Criteria

  • Code readability.
  • Design and usability.
  • Responsiveness (optional).
